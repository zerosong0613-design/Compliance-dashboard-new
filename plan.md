## Part 2. 7-Phase 연간 사이클

```
Phase 1: 리스크 식별       (1~2월)
  └─ 엑셀 업로드 → 배당 메일 → 현업 의견 → CP팀 확정

Phase 2: 리스크 평가       (2~3월)
  └─ A(발생가능성 1~5) × B(영향심각성 1~5) → X(고유리스크 VL~VH)

Phase 3: 통제수준 진단     (3~4월)
  └─ 현업 통제수단 입력 → CP팀 Y(효과성 VL~VH) 평가 → Z 자동 계산

Phase 4: 잔여리스크 확정   (4~5월)
  └─ Z 기준 관리대상 선정 → 사업팀 확인 → 최종 확정

Phase 5: 개선방안 도출·확정 (5~6월)
  └─ 현업 개선방안 제출 → CP팀 확정/반려 → Phase 5 완료

Phase 6: 개선방안 이행     (6~11월)
  └─ 확정 건만 이행 관리 → 월간 달성율 입력 → 평가

Phase 7: 점검·보고         (12월)
  └─ 연간 보고서 생성 → 다음 연도 준비
```
