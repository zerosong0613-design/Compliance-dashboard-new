# Phase 2 — 리스크 평가 (A·B 평가) 프로세스 설명서

> 최종 업데이트: 2026-03-25 (현재 구현 상태 기준)

---

## 1. 업무 흐름

```
Phase 1 완료 (리스크 식별 확정)
    ↓
[상황 A] 작년 엑셀에서 A/B/X 값이 이미 채워져 있음
  → Master "A·B 평가" 탭 열면 기존 값 표시
  → 변경 필요한 건만 수정 → 전체 저장 → Phase 2 확정

[상황 B] 리스크 목록만 들어왔고 A/B가 없음
  → A·B 평가 탭에서 전체 입력
  → X 자동 계산 (5×5 매트릭스)
  → 전체 저장 → Phase 2 확정

[공통] Phase 2 완료 확정 → Phase 3 활성화
```

## 2. 관련 페이지

| 페이지 | view ID / 탭 | 접근 권한 | 역할 |
|--------|-------------|----------|------|
| 리스크 Master > A·B 평가 탭 | `view-master` / `master-panel-assess` | CP팀 | A·B 입력, X 자동 계산 |

Phase 2는 별도 페이지가 아닌 **Master의 "A·B 평가" 탭**에서 처리합니다.
단계별 미션(phase2) 카드 클릭 시 `navigateTo('master') → switchMasterTab('assess')` 자동 전환.

## 3. 스코어링 상수

### A — 발생가능성 (5단계)

| 점수 | 레이블 | 색상 | 기준 |
|------|--------|------|------|
| 5 | 확신 | 빨강 | 과거 6개월 내 발생, 후속조치 미수행 |
| 4 | 높음 | 주황 | 과거 1년 내 발생, 후속조치 수행했으나 재발 가능 |
| 3 | 보통 | 황색 | 과거 3년 내 발생, 주기 모니터링 중 |
| 2 | 낮음 | 연초 | 과거 5년 내 발생, 근본원인 제거 완료 |
| 1 | 희박 | 녹색 | 발생 경험 없음 |

### B — 영향의 심각성 (5단계)

| 점수 | 레이블 | 법적 | 재무적 | 전략적 |
|------|--------|------|--------|--------|
| 5 | 매우 심각 | 대표자 처벌, 100억↑ 과징금 | 100억↑ 손실 | 면허 상실 |
| 4 | 상당한 영향 | 2년↑ 징역 / 2억↑ 벌금 | 50~100억 | 중대한 영향 |
| 3 | 보통의 영향 | 2년↓ 징역 / 2억↓ 벌금 | 10~50억 | 상당한 영향 |
| 2 | 낮은 영향 | 과태료, 10억↓ 과징금 | 1~10억 | 경미한 영향 |
| 1 | 미미한 영향 | 없음 | 1억↓ | 미미한 영향 |

### X — 고유리스크 매트릭스 (5×5)

```
X = X_MATRIX[A][B]

     B=1   B=2   B=3   B=4   B=5
A=5   M     H     VH    VH    VH
A=4   M     H     H     VH    VH
A=3   L     M     H     H     VH
A=2   VL    L     M     H     H
A=1   VL    VL    L     M     H
```

## 4. 주요 함수

| 함수명 | 역할 |
|--------|------|
| `renderMasterAssessTab()` | A·B 평가 탭 렌더링 (진행률 바 + X 분포 카드 + 평가 기준 접이식 + 테이블) |
| `updateRiskAB(riskId, type, val)` | A 또는 B 변경 시 X 자동 계산 + 부분 갱신 (전체 재렌더 방지) |
| `saveRiskAB(riskId)` | 개별 리스크 A·B 저장 |
| `saveAllRiskAB()` | 전체 리스크 일괄 저장 |

---

## 5. UI 특징

- **셀렉트 레이블**: 숫자만이 아닌 `3 보통`, `5 매우 심각` 등 의미 표시
- **평가 기준 접이식 패널**: A 기준 5단계 + B 기준 5단계 (법적/재무/전략적) 상시 참조
- **진행률 바**: 평가 완료 비율 + 미평가 건수 실시간 표시
- **X 분포 요약 카드**: VH/H/M/L/VL 건수 + 비율 시각화
- **미평가 건 강조**: 빨간 "미평가" 배지 + VH/H 행 배경색
- **부분 갱신**: A/B 선택 시 해당 행의 X 배지만 갱신 (550건에서도 성능 유지)

## 6. Phase 2 완료 조건

| # | 조건 | 판정 방식 |
|---|------|----------|
| 1 | A·B 평가 완료 90% 이상 | 자동 (assessed/total ≥ 0.9) |
| 2 | H·VH 등급 리스크 전수 평가 완료 | 자동 (모든 H/VH 건에 a_likelihood, b_impact 존재) |

> 100% 강제가 아닌 **90%** — 일부 미분류 리스크 허용. 단, H·VH는 반드시 전수 평가.
